**Cowrie SSH Honeypot**
This project is part of my MSc Cybersecurity practical learning. I set up a Cowrie SSH honeypot to simulate unauthorized access attempts and study attacker behavior in a controlled environment. The honeypot captures login attempts, commands executed by attackers, and interactions with a fake filesystem, which I then analyze to understand attack patterns and intrusion techniques.
